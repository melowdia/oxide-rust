[FR]

commands to reload via oxide

oxide.reload EpicLootCreate

admin command to force crate spawn
/spawncrate

the base crate spawns every 3 or 4 hours.

you can choose the number of bots that spawn by changing the number BotCount

private const int BotCount = 10; instead of 10 put the number you want

you can change the probability of bradley spawn by changing the line BradleyChance

private const float BradleyChance = 0.3f; changing the 0.3f to another number 0.xf currently 0.3f = 30%
